# K-Nearest-Neighbors

distance = √a^2 +√b^2 

<img src="https://www.mathsisfun.com/algebra/images/dist-2-points-b.gif"/>

a^2 + b^2 = c2


Start with:c2 = a2 + b2
Put in the calculations for a and b:c2 = (xA − xB)^2 + (yA − yB)^2
Square root of both sides:c = √(xA − xB)^2 + (yA − yB)^2
